# EE461-Project-V2

## Prerequisites
* [Python](https://www.python.org/downloads/)
* [node.js](https://nodejs.org/en)

## Starting the Backend
```
cd backend
python -m venv venv 
source venv/Scripts/activate
pip install -r requirements.txt --user
python app.py
```

## Starting the Frontend
```
cd frontend
npm install 
npm start
```
